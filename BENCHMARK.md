## 1. TOC 提取准确率

### 1.1 评估方法

**匹配规则：**
1. 标题文本相似度 > 0.8 (归一化编辑距离)
2. 页码误差 ≤ 2 页
3. 层级匹配 (Level 1/2/3)

**指标定义：**
- **Precision**: 提取的 TOC 中有多少是正确的
- **Recall**: 真实的 TOC 中有多少被提取了
- **F1**: Precision 和 Recall 的调和平均
- **Hierarchy Accuracy**: 层级关系正确的比例
- **Page Accuracy**: 页码完全匹配或误差≤2 页的比例

## 2. 检索准确率

### 2.1 评估方法

**测试集构建：**
- 每份文档 10 个 QA 对
- 问题类型：4 个事实性 + 4 个理解性 + 2 个推理性
- 人工标注正确答案所在的章节/页码

**指标定义：**
- **MRR (Mean Reciprocal Rank)**: 第一个正确答案的排名倒数均值 (越接近 1 越好)
- **Recall@K**: 前 K 个结果中有多少包含正确答案
- **NDCG@10**: 归一化折损累积增益 (排序质量指标)

## 6. 复现方法

### 6.1 运行完整评估

```bash
# 1. 构建测试集
python evaluation/dataset_builder.py

# 2. 创建 QA 标注模板
python evaluation/qa_dataset_builder.py

# 3. 人工标注 QA 测试集 (参考 evaluation/datasets/annotation_guide.md)

# 4. 运行评估
python evaluation/runner.py
```

### 6.2 单独运行压力测试

```bash
python evaluation/stress_test.py
```

### 6.3 查看测试集报告

```bash
# 测试集构成
cat evaluation/datasets/dataset_report.md

# QA 标注指南
cat evaluation/datasets/annotation_guide.md
```
